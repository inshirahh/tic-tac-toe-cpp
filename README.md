🎮 Tic Tac Toe Game (C++)


A simple and interactive Tic Tac Toe game built using C++.
This project demonstrates basic programming concepts like loops, arrays, and game logic.

🚀 Features

✅ 2D Array implementation

✅ Nested loops for board handling

✅ Interactive menu system

✅ Input validation

✅ Win & draw detection

✅ Clear game state display (Running / Win / Draw)

🎯 How to Play

Run the program

Choose Play Game

Players take turns:

Player 1 → X

Player 2 → O

Enter a number (1–9) to place your move

First player to align 3 symbols wins!


⚙️ How to Run

🔹 Using g++

g++ tic_tac_toe.cpp -o game

./game
